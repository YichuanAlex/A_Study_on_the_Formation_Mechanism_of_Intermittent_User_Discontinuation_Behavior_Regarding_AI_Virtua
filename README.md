# 公共图书馆 AI 虚拟馆员用户间歇性中辍行为形成机理研究

## 项目简介
本研究基于扎根理论和 fsQCA 方法，探讨公共图书馆 AI 虚拟馆员用户间歇性中辍行为的形成机理。

## 研究内容
- 通过深度访谈收集用户真实使用体验数据
- 运用三级编码（开放式编码、主轴编码、选择性编码）进行质性分析
- 采用 fsQCA 方法进行组态分析，识别影响用户间歇性中辍行为的关键因素组合

## 目录结构
- `mos_final/` - 最终版研究资料（论文、数据、图表等）
- `mos/` - 研究过程资料
- `mos_new/` - 新版分析资料
- `mos_origin/` - 原始研究资料
- `模拟对话数据/` - 访谈数据
